A Modern Cloudflare IP Speedtester for Windows

Finding the fastest Cloudflare IP for regions with severe internet censorship such as China can often feel like searching for a needle in a haystack.

The Problem

Cloudflare has massive IP ranges. Scanning all of them is impossible for a quick desktop utility. Most tools either:

  1. Scan sequentially (too slow).
  2. Use ICMP Pings (often blocked or prioritized differently than actual traffic).
  3. Have clunky, outdated generic UIs.

The Solution

I decided to build a standalone Windows application using Python and CustomTkinter to solve this.

Key Features

  • Material Design 3: Fits right into the modern Windows 11 aesthetic.
  • Smart Sampling: Instead of brute-forcing, it randomly samples IPs from the vast CIDR ranges.
  • Real TCP Testing: It performs actual TCP handshakes on port 443. This mimics real web traffic latency much better than a simple ping.
  • Concurrent Scanning: Uses multi-threading to check 50+ IPs simultaneously.

Under the Hood

The core logic revolves around a producer-consumer pattern.

  1. IP Generator: Parses ip.txt containing CIDR blocks and pushes random IPs into a queue.
  2. Thread Pool: A pool of workers picks up IPs and attempts a TCP connect.
  3. UI Controller: Updates the interface in real-time as results come in, maintaining a sorted “Top 5” list.
